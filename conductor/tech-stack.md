# Tech Stack: Schweitzer PTA Website

## Core Framework & Frontend
- **Framework:** [Next.js](https://nextjs.org/) (Version 16, utilizing the App Router)
- **Frontend Library:** [React](https://react.dev/) (Version 19)
- **Programming Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Version 4, for utility-first responsive design)

## Backend & Infrastructure
- **Backend-as-a-Service:** [Supabase](https://supabase.com/) (Used for Authentication and as a PostgreSQL Database)
- **Payment Processing:** [Stripe](https://stripe.com/) (For managing secure donations and sponsorships)
- **Deployment & Analytics:** [Vercel](https://vercel.com/) (For hosting and integrated performance analytics)

## Development & Testing
- **Testing Framework:** [Playwright](https://playwright.dev/) (For comprehensive end-to-end testing)
- **Linting & Formatting:** [ESLint](https://eslint.org/) (For code quality and consistency)
